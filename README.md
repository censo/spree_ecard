# SpreeEcard

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'spree_ecard'
```

Then execute:

    $ bundle

Or install it yourself as:

    $ gem install spree_ecard

Finally 

    $ rails generate spree_ecard:install

## Usage

Remember to set your eCard MerchantId and Password
